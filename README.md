# UrbanEdge Sales Analytics Dashboard 📊

## Overview

An interactive Power BI dashboard designed to analyze sales performance, profitability, customer behavior, product performance, regional trends, and business growth.

## Features

* KPI Cards (Revenue, Profit, Orders, Margin %)
* Regional Analysis
* Product Analysis
* Customer Analysis
* Forecasting & Trend Analysis
* Interactive Year Slicer
* Dynamic Filtering & Cross-Filtering
* Executive Summary Insights

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling

## Dashboard Pages

* Overview
* Regional Analysis
* Product Analysis
* Customer Analysis
* Forecasting

## Key Insights

* Identified top-performing products, customers, and regions.
* Analyzed revenue and profit trends.
* Evaluated customer segment performance.
* Generated business insights for decision-making.
